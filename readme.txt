This plugin changes the color of the help tab in wp-admin.

== Description ==

This plugin changes the color of the help tab in wp-admin. It also is part of the Admin Help Improvements project for Wordpress 3.8.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why would I want this?=

It's helps make the help button more visible in WP Admin.
